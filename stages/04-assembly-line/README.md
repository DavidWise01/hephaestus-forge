# Hephaestus 0.4 — Assembly Line

## Run
```bash
python selftest.py
python -m hephaestus.cli demo
python -m hephaestus.cli enqueue "create mesh monitoring dashboard"
python -m hephaestus.cli run
python -m hephaestus.cli dashboard
python -m hephaestus.cli export assembly_export.zip
```

Open `web/index.html`.

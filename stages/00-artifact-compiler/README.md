# Hephaestus 0.0 — Artifact Compiler

## Run

```bash
python selftest.py
python -m hephaestus.cli demo
python -m hephaestus.cli compile "detect fraud across a credential federation" --name Fraud_Scanner
python -m hephaestus.cli dashboard
python -m hephaestus.cli export hephaestus_export.zip
```

Open `web/index.html`.
